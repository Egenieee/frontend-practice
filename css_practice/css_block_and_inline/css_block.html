display: none vs visibility:hidden

해당 요소를 안보이게 하는 속성은 같지만, display: none으로 할 경우엔 해당 요소가 차지하는 공간마저 없애게 되지만, visibility:hidden으로 할 경우엔 보이지만 않게 할 뿐 해당 요소가 차지하는 공간은 그대로 남겨둔다.

주로 hidden을 많이 사용한다.
