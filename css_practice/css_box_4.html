- border-siziging을 content-box로 설정하게 되면 border나 padding을 제외하고 content 요소 자체의 너비, 높이를 지정된 크기로 지정하게 된다.

border-sizing: border-box;

- border-siziging을 border-box로 설정하게 되면 border나 padding을 포함한 박스 크기를 지정된 크기로 지정하게 된다. 모던 프론트웹에서는 border-box 속성을 사용한다.

HTML (the Hypertext Markup Language) and CSS (Cascading Style Sheets) are two of the core technologies for building Web pages. HTML provides the structure of the page, CSS the (visual and aural) layout, for a variety of devices.

- box-sizing은 상속되지 않는다. 디폴트 값은 content-box이기 때문에 전체 요소에 box-sizing을 border-box로 설정하기 위해 다음과 같은 css 설정을 한다.

*,
*::before
*::fter {
box-sizing: border-box;
}
