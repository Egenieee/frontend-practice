font-family

일반적으로 font-family에 여러 폰트를 설정해서 사용자 pc에 해당 폰트가 설치되어 있지 않을 경우를 대비한다.

세 가지 정도의 폰트를 설정하고 마지막 폰트를 generic-font를 설정한다.


웹폰트

웹폰트를 사용하게 되면, 사용자 pc에 해당 폰트가 설치되어 있지 않아도 어느 브라우저상에서나 해당 폰트로 웹페이지를 표현할 수 있다.

서버에서 따로 웹폰트를 다운받아야 하므로 느릴 수 있지만, 요즘 CDN 성능이 좋아지고 있어서 사용해도 괜찮다.
