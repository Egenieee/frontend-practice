letter-spacing & word-spacing 속성

글자 사이 간격은 letter-spacing 속성을 사용하고 단어 사이의 간격은 word-spacing으로 설정한다.

보통 px CSS 단위를 사용하여 설정한다. 설정 값을 키울수록 간격이 넓어진다. 음수도 속성 값으로 사용할 수 있다.

letter-spacing: 20px 글자 간격 20px

word-spacing: -3px 단어 간격 -3px

text-overflow 설정

문자열이 차서 넘칠 경우 (부모 영역을 벗어날 경우)에 자동 줄바꿈이 되지 않는 문자열 처리 설정

width 속성이 설정되어 있음, white-space가 nowrap으로 설정되어 있음, overflow 속성이 visible 이외의 값(hidden)으로 설정되어 있음 ⇨ 이 조건들이 설정되어 있는 상태에서 설정 가능.

즉 다시말해 width가 설정되어 있어서 overflow 인지를 체크할 수 있고, 자동 줄바꿈이 안되어 있고, 넘친 값이 hidden으로 보이지 않을 경우에 text-overflow 속성을 설정할 수 있다는 뜻

text-overflow 속성 값은 clip, ellipsis 가 있다.

text-overflow 속성 값을 ellipsis으로 설정할 시, 정해진 width 이후로는 ...으로 표시된다.
