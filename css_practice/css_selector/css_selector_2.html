css selector combination

p.color.font#genie {color: olive; font-size: 20px; text-align: center;}

- tag, class, id를 조합해서 selector로 사용가능하다. 클래스 마다 폰트 사이즈 지정, 폰트 컬러 지정, 정렬 지정 등을 해줄 때 이렇게 사용한다.

- 여러 class를 쓸 때는 태그 안에 "class= color font"와 같이 스페이스로 클래스 명을 이어준다.

class="color font" id="genie"

class="color"


복합 selector

- tag는 중첩시킬 수 있어, 각 태그 또는 요소 간에 부모/자식 관계가 생성된다.

- 복합 selector는 부모.자식 관계를 기반으로 특정 부분을 선택해 스타일을 적용할 수 있다.

- 다음과 같이 분류할 수 있다.

  • Descendant Selector : 스페이스로 표시
  • Child Selector : ">" 로 표시
  • Adjacent Sibling Selector : "+" 로 표시
  • General Sibling Selector : "~" 로 표시

div span {color: brown;}
