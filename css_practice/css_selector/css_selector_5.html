부정 셀렉터 (Negation Pseudo-Class Selector)

- 특정 태그를 제외한 태그에게만 스타일을 적용할 수 있다.

span:not(.genie) {color:olive;}


class="genie"
class="genieee"

정합성 체크 셀렉터 (Validity Pseudo-Class Selector)

- 태그가 정합성을 만족하는지 안하는지 여부에 따라 스타일을 적용할 수 있다.

1. required

input[type="text"]:invalid {background-color:brown;}
input[type="text"]:valid {background-color:olive;}

input type="text" id="id" name="id" required


2. pattern

input 태그에 입력한 값이 패턴에 만족해야만 valid한 상태라고 인식하고 스타일을 적용할 수 있다. * pattern 값은 정규표현식을 사용한다.

input[type="text"]:invalid {background-color:brown;}
input[type="text"]:valid {background-color:olive;}

input type="text" id="pw" name="pw" pattern="[0-9]{6}" required


가상 요소 셀렉터 (Pseudo-Element Selector)

선택한 요소의 특정부분을 선택하여 스타일을 적용할 수 있다.

h4::before { content: "[질문] "; color: olive; font-size: 1.5em;}
h4::after {content: " *구글 참조"; color: olive; font-size: 1.5em;}
h5::first-letter {font-size: 1.5em;}
h5::first-line {font-size: 1.3em;}
h5::selection {background-color: gray;}
