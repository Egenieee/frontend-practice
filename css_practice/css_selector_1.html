Css selector를 알아보자 !


tag selector

h4 {color : olve}

- 특정 태그에 스타일을 적용할 수 있다.

h4


class selector

.genie {color : olve}

- 특정 class에 스타일을 적용할 수 있다.

class="genie"


id selector

#genie {color : olve}

- 특정 id에 스타일을 적용할 수 있다.

id="genie"


attribute selector

[egnieee] {color : olive;}

- attribute selector에 속성을 주어 특정 속성을 가진 태그에게만 스타일을 적용할 수 있다

geniee

egenieee


[genie="ture"] {color : olive;}

- attribute selector에 속성 값을 주어 특정 값을 가진 태그에게만 스타일을 적용할 수 있다

genie="true"

genie="false"


[genie~="true"] {color : olive;}

- 속성 값 앞에 ~를 붙이면 속성 값에 공백을 두고 true라는 값이 있으면 스타일을 적용할 수 있다.

genie="truefalse"

genie="fasle true"


[genie|="true"] {color : olive;}

- 속성 값이 true이거나 뒤에 바로 "-" 문자가 오는 속성 값 이름일 경우에 스타일을 적용할 수 있다.

genie="true-true"

genie="falsetrue"


p[genie^="hello"] {color : olive;}

- p태그의 요소 중 속성 값이 hello로 시작하는 모든 태그에 스타일을 적용한다. 대소문자 구분 o

genie="helloworld"

genie="wordlhello"


p[genie^="world"] {color : olive;}

- p태그의 요소 중 속성 값이 world로 끝나는 모든 태그에 스타일을 적용한다. 대소문자 구분 o

genie="helloworld"

genie="wordlhello"


p[genie*="hi"] {color : olive;}

- p태그의 요소 중 속성 값에 hi 포함된 모든 태그에 스타일을 적용한다. 대소문자 구분 o

genie="everybody hi"

genie="aaaaaahiaaaaaaa"
