구조 가상 클래스 셀렉터 (Structured Pseudo-Class Selector)

- 부모 태그의 자식 태그 중 순서를 뽑아서 스타일을 적용할 수 있다.

p:first-child {color: red;}
p:last-child {color: purple;}
p:nth-child(2) {color: orange;}
p:nth-last-child(3) {color: brown;}

하지만 위와 같은 문법은 해당 태그 사이에 다른 태그가 있어도 카운트해서 스타일을 적용한다. 특정 태그 중에서만 순서를 뽑아서 스타일을 적용하고 싶을 때는 다음과 같은 셀럭터를 사용한다.

p:first-of-type {color: red;}
p:last-of-type {color: purple;}
p:nth-of-type(2) {color: orange;}
p:nth-last-of-type(3) {color: brown;}
