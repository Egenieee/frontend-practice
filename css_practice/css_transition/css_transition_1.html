CSS transition : CSS 속성 값 변경 시, 값 변화가 일정 시간 동안 일어나도록 해서 일종의 애니메이션 효과를 줄 수 있다.

  • transition-property
  • transition-duration
  • transition-timing-function
  • transition-delay
box-one : ease
box-two : linear
box-three : ease-in
box-four : ease-in-out
box-five : step-start
box-six : step-end
box-seven : steps(4, start)
box-eight : steps(2, start)
box-nine : steps(4, end)
