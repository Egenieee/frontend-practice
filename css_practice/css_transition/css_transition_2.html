CSS transition : CSS 속성 값 변경 시, 값 변화가 일정 시간 동안 일어나도록 해서 일종의 애니메이션 효과를 줄 수 있다.

  • transition-property
  • transition-duration
  • transition-timing-function
  • transition-delay
box-one : ease delay 0s
box-two : linear delay 1s
box-three : ease-in delay 2s
box-four : ease-in-out delay 3s

아래와 같이 단축 프로퍼티를 사용해서 한 줄에 표현할 수 있다. 순서만 지켜서 2개, 3개만 쓸 수도 있다.

transition : transition-property transition-duration transition-timing-function transition-delay
